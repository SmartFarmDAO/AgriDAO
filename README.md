## 🚀 Production Quick Start

### Prerequisites (Production Environment)
- **Node.js 18+** and npm (LTS version recommended)
- **PostgreSQL 14+** with extensions (PostGIS for location data)
- **Redis 6+** for caching and session management
- **Docker & Docker Compose** for containerized deployment
- **Firebase project** for push notifications and analytics
- **Cloud storage** (AWS S3, Google Cloud, or Azure)
- **SSL certificates** for production domains

### Development Setup

1. **Clone the repository**
```bash
git clone https://github.com/SmartFarmDAO/AgriDAO.git
cd agridao
```

2. **Install dependencies**
```bash
npm install
```

3. **Set up development environment**
```bash
cp .env.example .env
cp backend/.env.example backend/.env
# Edit environment files with your configuration
```

4. **Start development environment**
```bash
# Frontend development server
npm run dev                 # Starts on http://localhost:5173

# Backend services (in separate terminal)
cd backend
docker-compose up --build   # Starts backend on http://localhost:8000
```

5. **Verify installation**
```bash
# Run health checks
curl http://localhost:8000/health

# Run comprehensive tests
npm run test:e2e
./scripts/integration-test.sh
```

### Production Deployment

```bash
# Build and deploy to production
./scripts/deploy.sh deploy production blue-green

# Monitor deployment
./scripts/integration-test.sh health
docker-compose -f docker-compose.prod.yml logs -f

# Access production services
# Application: https://your-domain.com
# Admin Dashboard: https://your-domain.com/admin
# Monitoring: https://grafana.your-domain.com
```

VITE_API_URL=http://localhost:8000
