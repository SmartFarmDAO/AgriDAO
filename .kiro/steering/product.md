# AgriDAO Product Overview

AgriDAO is a decentralized agricultural platform for Bangladesh that connects farmers directly with buyers, eliminating middlemen and providing transparent, ethical financing.

## Core Features

- **Direct Marketplace**: Farmers list products, buyers purchase directly
- **Ethical Financing**: Interest-free funding through community donations and sponsorships
- **AI Advisory**: Weather alerts, market insights, yield predictions
- **Supply Chain Tracking**: Blockchain-based transparency from farm to consumer
- **DAO Governance**: Community-driven platform decisions and fund allocation
- **Multi-language**: Full Bengali and English support
- **Social Features**: Community posts, farmer profiles, networking

## User Roles

- **Farmers**: List products, manage inventory, receive funding, track orders
- **Buyers**: Browse marketplace, purchase products, track deliveries
- **Admin**: Moderate content, manage users, oversee platform operations
- **Community**: Vote on governance proposals, contribute to funding

## Technical Approach

- Monorepo structure with separate backend, frontend, blockchain, and mobile apps
- RESTful API with FastAPI backend
- React SPA frontend with TypeScript
- Ethereum smart contracts for governance and escrow
- PostgreSQL for relational data, Redis for caching
- Docker-based deployment
